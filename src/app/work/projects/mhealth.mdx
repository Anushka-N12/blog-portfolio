## Overview

An LLM-powered chatbot that refers to a database of QnAs of real therapist-patient conversations, giving comforting words & useful suggestions. 

## Technologies Used

- **RAG**: MongoDB for vector database, HuggingFace for embeddings. 
- **Next.js**: Web app.
